# gulp4 / Landing Page / WordPress

### Подтянуть все зависимости
`npm i`

### Собрать сборку   
`gulp build`


### Запустить  
`gulp`


### Удалить все изображения   
`gulp cleanimg`


### Генерировать сетку SmartGrid Дмитрия Лаврика   
`gulp grid`


### Генерировать Критические и Асинхронные стили
`gulp criticalStart`
